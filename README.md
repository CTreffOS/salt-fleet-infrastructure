# What is this?
This is a repository containing the Saltstack configuration for the laptop-fleet of the [Chaostreff Osnabrück](https://chaostreff-osnabrueck.de).

# Adoption
As the very first step, you have to be in our Hackspace Rabbithole and install the [salt-minion](https://docs.saltproject.io/salt/install-guide/en/latest/topics/install-by-operating-system/linux-deb.html) onto the target machine. Make sure to use some stock Debian with a minimal setup (meaning no desktop-environment).
Then you need to ask someone of our server-admins to accept the (new) machine key onto our master server. The machine will then be adopted shortly...
